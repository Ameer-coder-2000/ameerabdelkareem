- Responsive layout with left-side navigation
- Animated hero section with typed text
- Projects section with cards, thumbnails, and project details
- Skills section with icons, logos, and fallback visuals
- Contact section with email, phone, LinkedIn, and CV download

- Black and glass-style theme with a subtle water effect

## Tech Stack

- HTML
- CSS
- JavaScript
- Materialize CSS
- Font Awesome
- Typed.js

## Project Structure

- `index.html` - main portfolio page
- `assets/css/style.css` - custom styling
- `assets/img/` - images, logos, and project thumbnails
- `assets/vendor/typed.js/` - typed text animation library
- `Resume.pdf` - my CV
- `ameer.png` - profile photo

## Run Locally

You can open the site directly in a browser, or run a small local server:

```powershell
python -m http.server 8000
```

Then open:

```text
http://127.0.0.1:8000/
```

## Customize

- Edit `index.html` to change text, projects, skills, education, and contact details.
- Edit `assets/css/style.css` to change the theme, colors, spacing, and effects.
- Replace files inside `assets/img/` if you want new project images or logos.
- Replace `Resume.pdf` when you update your CV.
